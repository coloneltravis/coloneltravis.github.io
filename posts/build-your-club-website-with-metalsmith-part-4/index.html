Build Your Club Website with Metalsmith - Part 4

Build Your Club Website with Metalsmith - Part 4
Published: 12 December 2016
By Jake Bourne

_**In part 3 we built our club's website for the first time. However, although we could navigate between pages, it still looked like a website from the early 1990s.

In this part we will spruce up the website by applying some styles and colour.**_

Adding Some Styles

At the moment our website looks a little bland. Text is unformatted, the navigation bar is a vertical list of links and there is no colour. This can be solved by adding some styles using Cascading Style Sheets (CSS).

To give us a little extra help, we will download https://www.getbootstrap.com/. Bootstrap is a free collection of CSS styles which can be applied to tags to make a web page responsive to different devices like mobile phones, tablets, PCs or wide screen TVs.

  1. Download Bootstrap from https://www.getbootstrap.com/
  2. Extract the ZIP archive

  3. At the top level folder of myclubwebsite, create a new folder called, /assets

    /assets will be where we keep all third party components which we download. We will tell Metalsmith not to include these in the build process but to copy them instead.

  4. Go into the extracted archive and copy the bootstrap-3.3.2-dist to the /assets folder.

    The version number will likely be greater if you have downloaded a more recent version.

  5. Edit the top-level layout template, default.pug to include the bootstrap CSS file:-

    link(href='/assets/bootstrap-3.3.2-dist/css/bootstrap.min.css' rel='stylesheet' type='text/css')

  6. At the command prompt, install the metalsmith-static package:-

    npm install metalsmith-package --save

  7. Import the package by inserting the following line at the top of our build script:-

    var assets = require('metalsmith-static');

  8. Include the package in our build script to tell Metalsmith to copy the Bootstrap files as they are:-

    .use(assets( { src: 'assets', dest: 'assets' } ))
